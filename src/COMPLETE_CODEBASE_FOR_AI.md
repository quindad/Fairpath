## PROJECT OVERVIEW
- **Name:** Friend A Felon (acquired by FairPath Industries)
- **Purpose:** 5-sided reentry ecosystem connecting justice-impacted people with jobs, housing, resources, and community support
- **Tech:** React + TypeScript + Tailwind CSS + Supabase + SingleKey API
- **Design:** Black/charcoal with neon green (#A8F32C) accents, Futura Condensed Extra Bold font

## KEY FEATURES
1. **FastTrack Housing** - $75 ($65 with FairPath+) for guaranteed showing in 48 hours
2. **WOTC Jobs** - 27-question compliance with Save & Finish Later
3. **Free Marketplace** - 3-7 claims/month for donated items
4. **Gig Services** - Background-verified workers with escrow payments
5. **FairPath Score** - Universal 0-1000 scoring system
6. **FairPath+** - $2/month subscription ($10 off FastTrack, 7 claims/month)
7. **5 User Types** - Felons, Property Owners, Employers, Donors, Resource Partners
